api/GetId - получение id адресса для пользователя
api/GetQuestions - получение всех активных вопросов

api/GetAnswers - получение всех ответов
api/GetAnswersOnId/(id пользователя) - получение всех ответов опредленного пользователя
api/PostAnswers/(id вопроса)/(id ответа)/(id пользоватея) - отправка ответа, при успешном создании придет сам ответ

api/GetChoices/(id вопроса) - получения всех варинтов ответов для вопроса

при множественном ответе использовать api/PostAnswers для всех id ответов
